# Guide d'utilisation

Pour exécuter l'application, suivez ces étapes :

1. Positionnez-vous au niveau de la racine de ce projet.

2. Exécutez la commande suivante pour démarrer l'application : docker-compose up


Assurez-vous d'attendre que le processus se termine.

3. Une fois que le processus est terminé, ouvrez votre navigateur web.

4. Dans la barre d'adresse, entrez l'URL suivante :

localhost:80
ou simplement
localhost (puisque le port http c'est 80 ;) ) 

5. L'application devrait maintenant être accessible.

Pour vous connecter, utilisez les identifiants suivants :
- **Nom d'utilisateur :** smail
- **Mot de passe :** 123

Après la connexion, vous pouvez créer de nouveaux utilisateurs selon vos besoins.

Amusez-vous bien avec l'application ! Si vous avez des questions ou des problèmes, n'hésitez pas à nous contacter.
